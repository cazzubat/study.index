Teste título <h1>

A tag <h1> define o título principal de uma página ou seção, sendo crucial para a hierarquia de conteúdo e SEO.

exemplo:

Título teste

Parágrafo <p>

A tag <p> é usada para criar parágrafos de texto em HTML, organizando o conteúdo em blocos.

exemplo:

<p> teste parágrafo

Marcação <mark>

A tag <mark> é usada para destacar ou marcar texto importante em uma página, geralmente com um fundo amarelo.

exemplo:

<mark> teste marcação

Teste listas <ol> e <ul>

A tag <li> é usada para definir itens de lista dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>).

  1. <ol><li> teste lista ordenada
  • <ul><li> teste lista não ordenada

Seções em div e estilização em span

<div> é um elemento de bloco usado para estruturar seções grandes e quebra a linha.

<span> é inline, usado para estilizar partes pequenas do texto sem quebrar a linha.

exemplos:

<div> Olá mundo.
<span>Olá mundo.

os dois juntos:

<div>Olá <span>mundo.

Negrito em <b> e <strong>

<b> deixa o texto negrito apenas com efeito visual.

<strong> também deixa em negrito, mas dá ênfase semântica (mais importante para acessibilidade e SEO).

Itálico em <i> e <em>

<i> deixa o texto *em itálico* apenas com efeito visual.

<em> também deixa em itálico, mas com ênfase semântica.

Quebrar a linha <br>

A tag <br> é usada para quebrar a linha dentro de um parágrafo ou texto sem iniciar um novo bloco.

Sublinhar <u>

A tag <u> sublinha o texto, mas não tem significado semântico (só visual).

Atributo style

O atributo `style` é usado para aplicar CSS diretamente no elemento HTML de forma inline("em linha").

Atributo class

O atributo class é usado para aplicar estilos CSS através de uma classe, permitindo reutilizar o mesmo estilo em vários elementos. O símbolo . é usado no CSS para selecionar elementos pela classe (class).

exemplo:

Olá, sou classe.

Atributo id

O atributo id é usado para identificar unicamente um elemento na página e aplicar estilos ou manipulações específicas com CSS ou JavaScript. (O símbolo # é usado no CSS para selecionar um elemento pelo seu id)

exemplo:

Olá, sou um id.
